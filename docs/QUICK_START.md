# 🚀 Quick Start Guide

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/puneeth-webdev218/AI-FLASHCARD-TUTOR.git
   cd AI-FLASHCARD-TUTOR
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   streamlit run app.py
   ```

4. **Open in browser:**
   Navigate to `http://localhost:8501`

## Usage

### Generate Flashcards
1. Paste text or upload a PDF
2. Click "✨ Generate Flashcards"
3. Review and flip cards
4. Download as JSON (optional)

### Take Quiz
1. Generate at least 4 flashcards first
2. Go to "🎯 Take Quiz" tab
3. Answer MCQ questions
4. View your score

### Track Progress
1. Click "📊 Progress Dashboard" tab
2. View learning metrics and charts
3. Analyze performance by topic

## Features
- 📝 AI-powered flashcard generation
- 📄 PDF text extraction
- 🎯 Concept-based MCQ quizzes
- 📊 Progress tracking & analytics
- 💾 Offline operation (no API calls)

## Troubleshooting
- **No flashcards generated**: Provide longer, more detailed text
- **Quiz not available**: Generate at least 4 flashcards first
- **Dashboard empty**: Take at least one quiz
