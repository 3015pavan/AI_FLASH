===============================================================================
                        TROUBLESHOOTING GUIDE
===============================================================================

Problem 1: "Site can't be reached"
Solution:
  a. Verify server is running:
     Get-Process python | grep streamlit

  b. If not found, restart:
     cd "c:\Users\puneeth nagaraj\Downloads\AIML project\ai_flashcard_generator"
     python -m streamlit run app.py --server.port=8501

  c. Wait 5 seconds and try http://localhost:8501 again

Problem 2: Port 8501 Already in Use
Solution:
  Use alternative port:
  python -m streamlit run app.py --server.port=8502

  Then access: http://localhost:8502

Problem 3: PDF Text Not Extracting
Solution:
  Upgrade PyPDF2:
  pip install PyPDF2 --upgrade

  Some PDFs may need different tools (manual extract first)

Problem 4: Slow Generation
Solution:
  - This is normal on CPU (5-15 seconds)
  - Close other apps to free memory
  - For production: Install GPU drivers
  - Reduce flashcard count to speed up

Problem 5: Import Errors
Solution:
  Reinstall all dependencies:
  pip install -r requirements.txt --upgrade --force-reinstall
