🛠️ TROUBLESHOOTING GUIDE
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: ModuleNotFoundError (transformers, torch, etc.)
SOLUTION: pip install -r requirements.txt

PROBLEM: "Model not found at flashcard_t5"
SOLUTION: Run: python train.py (first time only, ~45 min)

PROBLEM: CUDA out of memory
SOLUTION: Edit train.py, change per_device_train_batch_size=8

PROBLEM: Slow generation (>30 seconds)
SOLUTION: 
  - Reduce num_beams: 6 → 4
  - Reduce num_flashcards: 10 → 6
  - Use GPU if available

PROBLEM: "Not enough context to generate flashcards"
SOLUTION: Use longer input text (>500 characters, >50 words)

PROBLEM: Low quality flashcards
SOLUTION:
  - Ensure input text is well-structured
  - Increase beam search: num_beams=8
  - Use more detailed, article-like text
